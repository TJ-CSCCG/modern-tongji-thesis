# 字体文件

本模板**不再**通过此目录分发字体文件。

- 默认字体集 `fandol`：从 [CTAN](https://mirrors.ctan.org/fonts/fandol.zip) 下载 OTF 文件
- Latin 衬线字体 `TeX Gyre Termes`：从 [CTAN](https://mirrors.ctan.org/fonts/tex-gyre-termes.zip) 下载 OTF 文件（用于中英文混排时 Latin 字符的渲染）
- 如需其他字体集（`founder`、`adobe`），请访问 [cjk-fonts-for-ctex](https://github.com/TJ-CSCCG/cjk-fonts-for-ctex)
- macOS / Windows 用户可直接使用系统字体（`fontset: "mac"` / `"windows"`），无需额外下载
- 详见 [README.md](../README.md#字体选择)
